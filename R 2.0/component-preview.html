1 · Alert

Inline status banner · 32px circular icon + tinted bg per variant

Warn variant

含 yellow tint + 黃色 icon container。用於需注意但非阻擋性提示，例如 IP 授權即將到期、低庫存。

Info variant

Blue tint。用於說明性訊息或補充說明，例如「手動補登的收入會標記為未驗證」。

Error variant

Red tint。用於失敗 / 同步異常 / 退款 / 爭議等錯誤狀態。

Success variant

Green tint。用於完成 / 結算成功 / Payout 已入帳等正向訊息。

Neutral variant (default)

No tint。最低調的提示，作為一般 callout 卡片使用。
