### D. Infrastructure（無視覺影響）

| 項目 | 動機 |

| Geist 字型從 Vercel Next.js node_modules 借用 | 沒專屬 Geist 下載點；Vercel 把 Geist 包進 `next/font/google` 的本地 cache。從 `Project/beamco-denise-lonely_react/node_modules/next/dist/next-devtools/server/font/geist-latin.woff2` 拷成 `Geist-Variable.woff2`，`geist-mono-latin.woff2` 拷成 `Geist-Mono.woff2`。Latin-only（沒 CJK）— CJK 走 Taipei Sans TC Beta fallback。 |

| theme.js click handler 改成只對 button/a preventDefault | 原本 `[data-theme-set]` click handler 一律 preventDefault，會阻止 radio input 的 native check 行為。Phase 2 加 radio-card UI 後發現 click radio card → theme 變了但 radio 沒被勾。改成只在 tagName === BUTTON / A 時 preventDefault，input/select 等表單控件保留 native 行為。讓 Phase 4 Settings Appearance section 可以直接用 radio + `data-theme-set` 而不需要額外 JS 同步。 |

## Convention

- **新加項目按 spec 章節編號 + 反饋日期累積**，最新在上
- **bug fix 不寫進這份檔案**（自己 CSS 寫錯、selector 衝突等都是內部除錯，commit message 即可）
- **撤除（C section）也要記**，方便日後追蹤「為什麼這條規則被拿掉」
