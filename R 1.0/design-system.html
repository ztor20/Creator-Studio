Adding a new icon · 新增 icon 流程

  1. 去 lucide.dev/icons 找到要的 icon，點開 → Copy SVG
  2. 在 icons.js 的 REGISTRY 加一行：'icon-name': '<path .../>',（只貼 <svg> 內層的 path / circle / rect 等，不含外層 <svg> wrapper）
  3. HTML 用 <i data-lucide="icon-name" class="ztor-icon"></i>

Customizing · 客製化

(1) CSS 改 size / color：.ztor-icon { color: var(--accent-yellow); }
(2) 單一 icon 覆寫 stroke：<i data-lucide="bell" stroke-width="2">
(3) Fork Lucide icon：抓 path 後改完，在 REGISTRY 用新名字註冊（如 bell-yellow），原版 bell 保留
(4) 完全自製：自己畫 SVG（24×24 viewBox），同樣寫進 REGISTRY
